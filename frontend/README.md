# Frontend - MERN Contact Manager

## Setup

1. `cd frontend`
2. `npm install`
3. `cp .env.example .env`
4. Adjust `VITE_API_URL` to backend URL (e.g. `https://<your-backend>/api`)
5. `npm run dev`

## Available scripts

- `npm run dev` - start development server
- `npm run build` - build production bundle
- `npm run preview` - preview built app

## Notes

- Uses React + Vite + Tailwind
- Functional components with hooks
- Supports search, add, edit, delete
